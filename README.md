# Rebol/AGG

AGG (Anti-Grain Geometry) extension for [Rebol3](https://github.com/Oldes/Rebol3) (version 3.14.1 and higher)

## Usage
```rebol
agg: import agg

```

## Extension commands:


#### `test` `:img`
Just an example
* `img` `[pair! image!]`

#### `draw` `:image` `:commands`
Draws scalable vector graphics to an image
* `image` `[image! pair!]`
* `commands` `[block!]`
